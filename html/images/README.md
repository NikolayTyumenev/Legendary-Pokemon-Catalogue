# Images Folder

Place all Pokemon sprite images here.

## Required Images:

For each Pokemon, you need:
- **Regular sprite**: `pokemon_name.png` (e.g., `mewtwo.png`)
- **Shiny sprite**: `pokemon_name_shiny.png` (e.g., `mewtwo_shiny.png`)
- **Alternate forms** (if applicable): `pokemon_name_form.png` (e.g., `mewtwo_mega_x.png`)

## Image Sources:

You can get Pokemon sprites from:
- **Serebii.net**: https://www.serebii.net/pokemon/
- **Bulbapedia**: https://bulbapedia.bulbagarden.net/
- **PokemonDB**: https://pokemondb.net/sprites

## Naming Convention:

Use lowercase with underscores:
- ✅ `articuno.png`
- ✅ `articuno_shiny.png`
- ✅ `giratina_origin.png`
- ❌ `Articuno.PNG`
- ❌ `articuno-shiny.png`

## Total Images Needed:

For ~80-95 Legendary/Mythical Pokemon:
- Minimum: 160-190 images (regular + shiny for each)
- With alternate forms: 200-250 images
