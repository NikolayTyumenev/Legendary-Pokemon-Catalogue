# Pokemon Images

Place Pokemon sprite images in this directory.

## Naming Convention: 
- Regular: `pokemonname.png` (e.g., `mewtwo.png`)
- Shiny: `pokemonname_shiny.png` (e.g., `mewtwo_shiny.png`)
- Forms: `pokemonname_formname.png` (e.g., `mewtwo_mega_x.png`, `mewtwo_mega_y.png`)

## Recommended Image Specifications:
- Format: PNG with transparency
- Size: 200x200 pixels minimum
- Quality: High resolution official sprites preferred

## Data Source:
Images sourced from Serebii.net
